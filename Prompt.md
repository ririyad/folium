## Iteration 0.1

> Insight: This app will be a record of tree plantation activity. The home page is a dashboard, where Total Tree Count, Active Volunteers, Areas Coverd number along with other insights will be shown

Nagigational Menu:
- Home Page: For now, it will have a simple Welcome Message
- Tree Management: Tree CRUD
    - Name,
    - Species,
    - Purchase Date
    - Lat/Long
    - Price 
    - Status (Healthy, Growing, Needs Care)
- Contribution Insights
    - Monthly Plantation Progress
    - Species Distribution
    - Volunteer Contrubution

### Tech Stack

    - Sveltekit
    - Netlify
